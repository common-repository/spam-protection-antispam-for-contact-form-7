Powerful anti-spam plugin for Contact Form 7. No comment spam, no user registration spam, no contact form spam, Easy to install, Easy to use.

== Description ==

This plugin is a WordPress Anti Spam for Contact Form 7 that protects your website from spam without the use of any captcha. It gives you 100% protection against spambots. It is simple to install and is very easy to use. It sends daily and weekly spam reports so you are kept up to date with the status of your website. 

**FEATURES**
* Stops spam in comments
* Stops spam for registration forms
* Stops spam in contact forms.
* Stops spam in Search forms.
and much more
